## 常用正则表达式

### 1. 只允许匹配非中文字符

```js
function onlyContainsNonChinese(s) {
  return /^[^\u4e00-\u9fa5]+$/.test(s);
}

console.log(onlyContainsNonChinese('你好，世界!'));  // false
console.log(onlyContainsNonChinese('Hello, 世界!'));  // false
console.log(onlyContainsNonChinese('Hello, world!'));  // true

// form rules
{
  pattern: /^[^\u4e00-\u9fa5]+$/,
  message: '不能输入中文',
  trigger: 'change',
}
```
